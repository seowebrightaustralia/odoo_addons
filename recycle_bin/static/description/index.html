Recycle Bin

Trash management for deleted data history, include the screenshot of the data and ability to restore deleted data

Overview

One of the most significant concerns with Odoo is that there is no history of deleted data, which frequently causes problems in Odoo implementation with users believing that data has mysteriously disappears. To overcome this issue, this module functions as recycle bin, storing a history of data deletion as well as saving screenshots when the user deletes data and having the ability to restore deleted data.
